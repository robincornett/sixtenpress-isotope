SixTen Press Isotope is a little plugin which makes setting up an isotope based archive easy peasy.

== Description ==
This plugin doesn't do anything until you tell it to. Go to Settings > 6/10 Press Isotope and pick where you want the effect to happen.

= Credits =
Built by <a href="https://robincornett.com/">Robin Cornett</a>

== Installation ==
1. Upload the entire `sixtenpress-isotope` folder to your `/wp-content/plugins` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Visit the Settings > 6/10 Press Isotope page to enter your license key and enable the stylesheet.
1. Add widgets to your site!
